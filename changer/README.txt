-> This app hides a smaller image in a bigger one.

-> On each folder, the first image is the one that is read.

		--- Encoding photo ---
1- Put the image to hide in the folder ImageToEncode
2- Put the cover image in the folder CoverImages
3- Run the PhotoEncoder application. 
4- A photo named "secret.png" will be generated

		--- Decoding photo ---
1- Put the image to decode in the folder ImageToDecode
2- Run the PhotoDecoder application
3- A photo named "reveal.png" will be generated
